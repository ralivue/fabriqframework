# Fabriq Framework

___This project is no longer maintained or updated and is archived. Should you choose to fork it, please maintain the proper license and credits.___

Fabriq is an object-oriented, MVC, rapid development framework for PHP. Fabriq gives you an MVC architecture for developing pages as well as creating modules.

This page will also list modules that are not part of the core module set as they are developed to help extend your Fabriq projects.
